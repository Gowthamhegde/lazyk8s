# GitHub Repository Setup Guide

Add this as your repository description on GitHub:

```
⎈ A stylish, interactive Kubernetes TUI — see everything in your cluster without running individual kubectl commands
```

**Topics to add:**
- kubernetes
- kubectl
- tui
- terminal
- devops
- python
- textual
- k8s
- cluster-management
- developer-tools

Consider adding:
- Automated testing on push
- Release automation
- Code quality checks (linting)

## Social Preview Image

Upload one of your screenshots as the social preview image:
1. Go to Settings → Options
2. Scroll to "Social preview"
3. Upload `screenshots/1.png` or create a custom banner

## Enable Discussions (Optional)

Enable GitHub Discussions for:
- Q&A
- Feature discussions
- Show and tell
- General community chat

Go to Settings → Features → Check "Discussions"
